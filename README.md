# Portfolio personnel - Zayd BENZAKOUR

Portfolio personnel développé afin de présenter mon profil, mon parcours, mes compétences et mes projets en informatique.

Ce site a été réalisé dans le cadre de ma progression en BUT Informatique, avec l’objectif de construire progressivement une présence professionnelle en ligne et de mettre en avant mes apprentissages.

## Objectifs du projet

Ce portfolio me permet de :

- présenter mon profil de manière claire et professionnelle ;
- regrouper mes projets personnels et universitaires ;
- mettre en avant mes compétences techniques ;
- intégrer mon CV ;
- pratiquer la structuration d’un projet web ;
- améliorer mes bases en HTML, CSS et JavaScript ;
- préparer un futur déploiement sur un VPS avec nom de domaine.

## Technologies utilisées

- HTML
- CSS
- JavaScript
- Git
- GitHub

## Fonctionnalités

Le site contient actuellement :

- une section d’accueil avec une présentation rapide ;
- une section à propos ;
- une section compétences ;
- une section projets ;
- une section parcours ;
- une section contact ;
- un lien vers mon CV ;
- une image de profil ;
- un logo personnel ;
- un mode clair / sombre ;
- un bouton de retour en haut de page ;
- des animations simples au chargement et au défilement ;
- une structure responsive adaptée aux différentes tailles d’écran.
